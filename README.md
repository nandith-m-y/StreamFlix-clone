# 🎬 StreamFlix – Premium Streaming Platform Landing Page

A modern, responsive streaming platform landing page inspired by popular streaming services, built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project showcases a premium user interface with responsive layouts, interactive components, smooth animations, and a professional design suitable for frontend portfolio projects.

## 📖 Project Overview

StreamFlix is a frontend-only streaming platform homepage created as part of a Frontend Developer Internship UI Clone Challenge. The website is fully responsive and provides an engaging user experience with sections commonly found in premium streaming platforms.

The project focuses on clean UI design, responsive layouts, reusable code, and modern frontend development practices.

## 🚀 How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/nandith-m-y/NetFlix-clone.git
```

### 2. Open the Project Folder

```bash
cd NetFlix-clone
```

### 3. Launch the Website

Open `index.html` in your browser

OR

Use the **Live Server** extension in Visual Studio Code.
