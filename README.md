# 🎬 StreamFlix – Premium Streaming Platform Landing Page

A modern, responsive streaming platform landing page inspired by popular streaming services, built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project showcases a premium user interface with responsive layouts, interactive components, smooth animations, and a professional design.

## 📖 Project Overview

StreamFlix is a premium streaming platform landing page developed as part of a Frontend Developer Internship UI Clone Challenge.

The project recreates the experience of a modern streaming platform while using only frontend technologies. It demonstrates responsive web design, clean UI, interactive components, and modern development practices.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/nandith-m-y/StreamFlix-clone.git
```

Open the project folder:

```bash
cd StreamFlix-clone
```

Open **index.html** in your browser or run it using the **Live Server** extension in Visual Studio Code.
